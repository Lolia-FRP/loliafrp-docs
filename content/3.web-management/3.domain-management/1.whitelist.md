## 域名白名单
在输入框中输入目标域名，要求为合法的二级域名，且不包含协议部分（如 http:// 或 https://）。例如 `example.com` 

## 已添加域名
- 此处展示已经添加到白名单中的域名列表，包含域名和状态等信息 
- 在上方添加完成域名后，会展示需要添加的验证信息，请将验证信息添加到对应域名的 DNS 记录中，记录类型为 TXT，等待 DNS 记录生效后点击刷新验证按钮，验证成功后该域名就可以在创建 HTTP 隧道时选择了。  
- 在完成域名验证后，域名状态会变更为“已生效”，同时在域名后方会显示该域名是否已备案的标记。当前请联系管理员手动进行备案审查，只有标记为“已备案”的域名才能在中国大陆节点上创建 HTTP 隧道。（注意：部分节点需要特定接入商的备案，此处仅表示该域名已在工信部登记备案）
