# MCSManager FRP模板

- MCSManager FRP模板是由 [语千](https://github.com/yuqian114514) 基于各大FRP厂商的命令行工具和 [MCSManager](https://github.com/MCSManager/MCSManager) 面板制作

# MCSManager FRP模板部署

## 部署MCSManager

没有部署MCSManager可前往 [https://docs.mcsmanager.com/zh_cn/](https://docs.mcsmanager.com/zh_cn/) 查看教程部署

## 安装模板

> 💡 *如果已经安装过可跳过*

1. 前往[https://gitee.com/yuqian114514/mcsm-frp-market/releases/latest/](https://gitee.com/yuqian114514/mcsm-frp-market/releases/latest/)下载 market.json 文件保存
2. 前往您的 MCSManager 面板登录
3. 在面板侧边栏或顶部导航中，找到并点击 **「模版市场」** 
4. 点击 **「编辑应用市场」** 按钮
5. 点击 **「操作」** → 选择 **「清空当前模板」** → 弹出确认框后点击 **「确认」** 
6. 点击 **「导入」**→ 选择 **「从文件导入」** → 选择刚刚下载的 market.json 文件
7.  点击 **「保存到面板」**

## 安装Lolia CLI模板

1. 在模板市场找到 FRP 专区并选择你的服务器
2. 找到 LoliaFRP-CLI 模板安装

## 启动Lolia CLI

1. 在 Lolia FRP 控制台创建好面板后，点击复制 `连接配置` 中的 `使用 LoliaFRP-CLI 快捷启动` 命令行，如

 ```shell
 ./frpc -t 114:abcdefgjhijklmnopqrstuvwxyz123456
 ```

  Windows 版的我们需要改一下命令
  Windows：

 ```shell
 frpc.exe -t 114:abcdefgjhijklmnopqrstuvwxyz123456
 ```
2. 打开你的 MCSManager 面板，找到你刚安装的 Lolia-CLI 模板
3. 在应用实例控制台找到`功能组`中的`应用实例设置`
4. 把`启动命令`中写的提示删掉，把你复制的`LoliaFRP-CLI 快捷启动命令`粘贴在启动命令里，点击保存
5. 开启您的实例
6. 连接成功后会显示类似如下的日志输出：

```shell
2:25AM INFO <client/control.go:205> LoliaFRP-CLI: [1145141919810] [1.abcdefgjhijklmnopqrstuvwxyz123456] 您可通过 114514.com:1145 访问您的服务
```

7. 您可通过 `114514.com:1145` 访问您的服务了，恭喜您成功使用 LoliaFRP-CLI 启动了 LoliaFRP！
