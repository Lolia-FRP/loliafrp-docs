## 概述

LoliaFRP Open API 支持标准的 **OAuth2 Authorization Code** 授权码模式

::callout{icon="i-heroicons-shield-check" color="green"}
**安全优势**：使用 OAuth2，用户可以随时撤销对第三方应用的授权，且第三方应用永远无法获取用户的登录凭证
::

## 端点信息

| 端点 | 地址 |
| --- | --- |
| 授权端点 | `https://dash.lolia.link/dash/oauth/authorize` |
| Token 端点 | `https://api.lolia.link/api/v1/oauth2/token` |


## 权限范围（Scope）

| Scope | 说明 |
| --- | --- |
| `user:read` | 读取用户基本信息 |
| `tunnel:read` | 读取隧道信息 |
| `tunnel:write` | 创建/修改/删除隧道 |
| `traffic:read` | 读取流量统计 |
| `domain:read` | 读取域名信息 |
| `domain:write` | 创建/修改/删除域名 |
| `node:read` | 读取节点信息 |
| `all` | 完全访问权限 |

::callout{icon="i-heroicons-exclamation-triangle" color="amber"}
**注意**：`all` 权限将授予应用完全访问权限，请谨慎使用
::
