## 什么是 LoliaFRP Open API？

- LoliaFRP Open API 是 LoliaFRP 官方提供的开放接口，允许开发者通过编程方式与 LoliaFRP 平台进行交互
- 您可以使用 API 来管理隧道、查询流量统计、获取节点信息等，非常适合自动化运维或开发第三方客户端

## 鉴权方式

**我们推荐使用 OAuth2 进行 LoliaFRP Open API 鉴权**

### OAuth2 鉴权（推荐）

OAuth2 是最安全且推荐的鉴权方式，适用于第三方应用接入 LoliaFRP 平台

::callout{icon="i-heroicons-check-circle" color="green"}
**推荐理由**：OAuth2 可以让用户安全地授权第三方应用访问其 LoliaFRP 账户，无需直接暴露用户凭证，同时支持细粒度的权限控制
::
